**思路**：找到此图最小生成树；再遍历最小生成树中的所有边，寻找修改前后差值最大的边。

**最小生成树**：克鲁斯卡尔算法
1. 首先将所有边从小到大排列依次遍历，若此边的两个端点从属同一个集合就不选此边，否则就选。
2. 怎样判断两点在一个集合——并查集
- 先用一个p数组，形如：`p = [0, 1, 2, 3, 4]`。代码：`p = [i for i in range(n+1)]`
含义是第i个位置的父亲是`p[i]`。此时每个点的父亲都是它自己，意味每点都是一个独立的集合。
- 当选中一条边时，检查此边的两端点x、y是否属于同一集合。代码：`if (p[x] != p[y])`
- ！！！此时出现一个这样的问题：若`p[2] = 3, p[3] = 3, p[4] = 2`，要检测的是（3， 4）这条边，按我们上述检测方法（3， 4）是可以选的，但其实他们已经在同一集合了。_（因为4的父亲是2,2的父亲又是3，所以4的祖先是3，与3的祖先是同一个，所以3、4是属于同一集合的）
- 经过分析我们可知，其实我们真正要检测的不是x、y两点的父亲是否一样，而是x、y的祖先是否一样。
- 怎么追溯一点的祖先呢——递归。所以判断标准应改为`if find(x) != find(x)`
```python
def find(x){
	if x == p[x]:             //当自己就是自己的祖先时，就说明找到了这个集合的头，可以返回了
		return p[x]
	p[x] = find(p[x])         //一直递归让x的父亲变成他父亲的父亲，最终变成这个集合的头
		return p[x]           //返回这个集合的头，即x的祖宗
}
```
3. 将所有选中的边加入`ans[]`数组，就是找到的最小生成树

最后遍历ans数组，求所有边长的总和，并找`边长-x&y`最大的，用边长总和-最大差值。

**代码**py：时间复杂度O(nlogn)
```Python
def find(x):
    if x == p[x]:
        return p[x]
    p[x] = find(p[x])
    return p[x]

n, m = list(map(int, input().split()))
a = []
for i in range(m):
    a.append(list(map(int, input().split())))
a.sort(key = lambda x:(x[2]))

p = [i for i in range(n+1)]
cnt = 0
ans = []
for i in a:
    x, y = i[0], i[1]
    X, Y = find(x), find(y)
    if X != Y:
        p[X] = Y
        cnt += 1
        ans.append(i)
    if cnt == n-1: break
    
maxx, summ = 0, 0
for i in ans:
    x, y = i[0], i[1]
    summ += i[2]
    if i[2]-(x&y) > maxx:
        maxx = i[2]-(x&y)

print(summ-maxx)
```
